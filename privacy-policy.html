Gizlilik Politikası

Canlı Türkçe Altyazı Chrome Eklentisi · Son güncelleme: Mart 2026

1. Genel Bakış

Canlı Türkçe Altyazı ("Eklenti"), İngilizce konuşmayı gerçek zamanlı olarak Türkçe altyazıya çeviren bir Chrome tarayıcı eklentisidir. Bu gizlilik politikası, eklentinin hangi verileri işlediğini ve bu verilerin nasıl kullanıldığını açıklar.

2. Toplanan ve İşlenen Veriler

Eklenti aşağıdaki verileri yalnızca işlevselliğini sağlamak amacıyla işler:

  • Ses verisi (mikrofon): Mikrofon modu seçildiğinde, tarayıcının Web Speech API'si aracılığıyla ses veriniz işlenir. Bu ses verisi yalnızca tarayıcı içinde kalır ve herhangi bir sunucuya gönderilmez.
  • Ses verisi (sekme sesi): Hoparlör/sekme modu seçildiğinde, sekme sesi kaydedilir ve konuşmayı metne çevirmek için Groq Whisper API'sine gönderilir.
  • Metin verisi: Tanınan İngilizce metin, Türkçeye çevrilmek üzere Google Translate API ve/veya MyMemory API'sine gönderilir.
  • Kullanıcı ayarları: Altyazı konumu, yazı boyutu gibi tercihler tarayıcının yerel depolamasında (chrome.storage.local) saklanır.
  • API anahtarı: Kullanıcı tarafından girilen Groq API anahtarı, tarayıcının yerel depolamasında saklanır.

3. Üçüncü Taraf Hizmetleri

Eklenti aşağıdaki üçüncü taraf hizmetlerini kullanır:

  • Google Translate API (translate.googleapis.com) – Metin çevirisi
  • MyMemory Translation API (api.mymemory.translated.net) – Yedek çeviri hizmeti
  • Groq Whisper API (api.groq.com) – Sekme sesi modu için konuşmadan metne dönüşüm

Bu hizmetlere gönderilen veriler, ilgili hizmetlerin kendi gizlilik politikalarına tabidir. Eklenti geliştiricisi bu verileri toplamaz, saklamaz veya üçüncü taraflarla paylaşmaz.

4. Veri Saklama

Eklenti herhangi bir sunucu veya veritabanı kullanmaz. Tüm veriler anlık olarak işlenir ve kalıcı olarak saklanmaz. Kullanıcı tercihleri ve API anahtarı yalnızca kullanıcının kendi tarayıcısının yerel depolamasında tutulur.

5. Veri Paylaşımı

Eklenti, yukarıda belirtilen çeviri ve transkripsiyon hizmetleri dışında hiçbir veriyi üçüncü taraflarla paylaşmaz. Kişisel bilgileriniz toplanmaz, satılmaz veya pazarlama amaçlı kullanılmaz.

6. İzinler

Eklentinin kullandığı Chrome izinleri ve nedenleri:

  • activeTab: Aktif sekmeye altyazı overlay'i eklemek için
  • scripting: Altyazı bileşenlerini sayfalara enjekte etmek için
  • storage: Kullanıcı ayarlarını kaydetmek için
  • tabs: Aktif sekme bilgisine erişmek için
  • offscreen: Sekme sesini arka planda yakalamak için
  • tabCapture: Sekme ses akışına erişmek için

7. Kullanıcı Hakları

Kullanıcılar her zaman:

  • Eklentiyi kaldırarak tüm yerel verileri silebilir
  • Mikrofon iznini tarayıcı ayarlarından iptal edebilir
  • Sekme sesi yakalama özelliğini kullanmayı tercih etmeyebilir

8. Değişiklikler

Bu gizlilik politikası zaman zaman güncellenebilir. Önemli değişiklikler eklenti güncellemeleriyle birlikte duyurulacaktır.

9. İletişim

Gizlilik politikasıyla ilgili sorularınız için GitHub Issues sayfasından iletişime geçebilirsiniz.
